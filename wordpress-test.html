WordPress Site - E1 Calculator Widget Demo

Lämpöpumpun säästölaskuri

Julkaistu: Tammikuu 2025 | Kategoria: Energia

Tässä artikkelissa voit laskea, paljonko säästät lämpöpumpulla. Alla on interaktiivinen laskuri, joka auttaa sinua arvioimaan säästöpotentiaalin.

Huom! Tämä on E1 Calculator Widget -testisivu. Widget latautuu automaattisesti alla olevaan elementtiin.

Miten widget toimii?

E1 Calculator Widget on itsenäinen JavaScript-paketti, joka voidaan upottaa mihin tahansa WordPress-sivulle. Se sisältää kaikki tarvittavat komponentit ja tyylit.

Asennus WordPress-sivulle

Lisää seuraava koodi WordPress-sivullesi tai artikkeliisi:

<!-- Lisää widgetin tiedostot -->
<link rel="stylesheet" href="/path/to/e1-calculator-widget.min.css">
<script src="/path/to/e1-calculator-widget.min.js"></script>

<!-- Lisää widget-elementti -->
<div id="e1-calculator-widget"
    data-supabase-url="YOUR_SUPABASE_URL"
    data-supabase-key="YOUR_SUPABASE_ANON_KEY"
    data-max-width="1200"
    data-show-visual="true">
</div>

Vaihtoehtoisesti JavaScript API:lla

<script>
  // Alusta widget manuaalisesti
  E1Calculator.init('widget-container-id', {
    supabaseUrl: 'YOUR_SUPABASE_URL',
    supabaseAnonKey: 'YOUR_ANON_KEY',
    maxWidth: 1200,
    showVisualSupport: true,
    customTokens: {
      // Voit ylikirjoittaa design tokeneja
      colors: {
        brand: {
          primary: '#your-brand-color'
        }
      }
    }
  });
</script>

Ominaisuudet

  • ✅ Täysin itsenäinen - ei riippuvuuksia Next.js:stä
  • ✅ Sisältää kaikki design tokenit JSON-konfiguraatiosta
  • ✅ Responsiivinen mobiili- ja työpöytänäkymä
  • ✅ Supabase-integraatio datalle
  • ✅ Progressiivinen lomake kortteina
  • ✅ Visuaalinen tuki paneeli
  • ✅ Teemojen tuki

Kustomointi

Widget käyttää design tokeneja, jotka on määritelty cardstream-complete-config.json tiedostossa. Voit ylikirjoittaa näitä tokeneja customTokens parametrilla.
