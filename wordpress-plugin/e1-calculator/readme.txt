Embed Energiaykkönen's heat pump savings calculator on your WordPress site with a simple shortcode.

== Description ==

The E1 Calculator plugin allows you to easily embed Energiaykkönen's professional heat pump savings calculator on your WordPress website. Help your visitors calculate their potential energy savings and environmental impact with just a few clicks.

**Features:**

* Simple shortcode integration [e1_calculator]
* Automatic height adjustment for seamless integration
* Responsive design that works on all devices
* Customizable height and width options
* Clean, professional interface
* No coding knowledge required

**Benefits for Your Visitors:**

* Calculate annual energy savings
* Estimate CO2 reduction
* See 5 and 10-year savings projections
* Get personalized recommendations
* Receive detailed PDF reports

== Installation ==

1. Upload the `e1-calculator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > E1 Calculator to configure the plugin
4. Add the shortcode `[e1_calculator]` to any page or post

**Manual Installation:**

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Go to Plugins > Add New > Upload Plugin
4. Choose the ZIP file and click "Install Now"
5. Activate the plugin after installation

== Frequently Asked Questions ==

= How do I add the calculator to my page? =

Simply add the shortcode `[e1_calculator]` to any page or post where you want the calculator to appear.

= Can I customize the calculator's appearance? =

Yes! You can customize the height, width, and add custom CSS classes:
* `[e1_calculator height="800"]` - Set custom height in pixels
* `[e1_calculator width="100%"]` - Set custom width
* `[e1_calculator class="my-custom-class"]` - Add custom CSS class

= Does the calculator work on mobile devices? =

Yes, the calculator is fully responsive and works perfectly on all devices including smartphones and tablets.

= Is the calculator available in multiple languages? =

Currently, the calculator is available in Finnish. Additional languages may be added in future updates.

= How does the automatic height adjustment work? =

The plugin uses JavaScript to automatically adjust the iframe height based on the calculator's content, ensuring no scrollbars appear and all content is visible.

= Can I use multiple calculators on the same page? =

Yes, you can add multiple calculator instances on the same page. Each will work independently.

= Is the data secure? =

Yes, all data is processed securely. Form submissions are handled with proper validation and sanitization.
